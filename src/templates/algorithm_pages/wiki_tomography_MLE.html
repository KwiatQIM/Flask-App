tomography_MLE

This Maximum Likelihood Estimation approach to quantum state tomography calculates the state that has the highest probability of producing the observed measurements

Inputs:

  • Starting Density matrix
    • The Density Matrix that is used to start the Maximum Likelihood Estimation
  • Coincidences
    • The array of coincidence counts for each measurement basis.
  • Measurement Density Matrices
    • A 3-D array containing the Density Matrices for each measurement basis.
  • Accidentals
    • An array containing the predicted amount of accidental counts for each measurement state.
  • Overall Norms
    • An array containing the relative weights for each measurement, used for drift correction.

Step 1:

The starting density matrix is converted to its corresponding T values

The T values are used in to adjust the state to obtain the most likely option.

Step 2:

The function loops through different t values, and uses least squares regression to find the T values that are most likely to have produced the observed output.

Step 3:

The density matrix of the most likely state is reconstructed from the T values.

Return Values:

  • Final Density Matrix
    • The Density Matrix corresponding the the state most likely to have produced the observed measurements.
  • Intensity
    • The value used to normalize the state to trace 1 after optimization.
  • Final Optimization Value
    • The final value of the internal optimization function
